How To Run
--------------------------------------------------------------------
Use following command to run 

$ hadoop jar Problem1.jar UniqueCategories /dataset/business.csv /sns150630/output


input file: /dataset/business.csv
Output directory: /sns150630/output

----------------------------------------------------------------------

Output of the program is there in a "Output" file. You can use Notepad++ to open the
same file to see the output.
